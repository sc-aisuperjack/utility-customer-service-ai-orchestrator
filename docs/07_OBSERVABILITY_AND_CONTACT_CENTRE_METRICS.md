# Observability and Metrics

Log conversation_id, channel, prompt_version, model_id, intent, risk_flags, retrieved docs, tool calls, handoff reason, latency, token usage and output safety.

Metrics:
containment, deflection, FCR, CSAT, AHT, repeat contact, groundedness, hallucination rate, tool success, P95 latency, cost per resolved contact.

Containment alone is not success.
