# Glossary

Agent: LLM system that can choose tools/steps.
Workflow: developer-defined sequence.
RAG: retrieval-augmented generation.
Groundedness: answer supported by sources.
Tool calling: structured API/function invocation.
Guardrail: safety/compliance control.
Prompt injection: attempt to override instructions.
ASR: speech recognition.
TTS: text to speech.
DTMF: keypad input.
Barge-in: caller interrupts bot.
Containment: resolution without human.
FCR: first-contact resolution.
CSAT: customer satisfaction.
AHT: average handle time.
Canary: limited rollout.
